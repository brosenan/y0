# $D_0$: A Language for Defining Dynamic Semantics

This is a language spec for the $D_0$ language, a language for defining the
_dynamic semantics_ of programming languages. It is provided here as an example
of how to define a language using $y_0$, and is validated against its $y_0$
definition in [d0.y0](../d0.y0).
